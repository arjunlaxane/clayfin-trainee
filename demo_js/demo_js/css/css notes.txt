Cascading: one inside another

style: color, appearence, layout, height, font size ,width

Sheet : file



inline -> more priority - only for that html element - cannot be resused for another element
embedded -> least - only by that html file alone, cannot be reused by other html pages
external -> less - can be used by any html files (global)

how to add:

inline styles:

using attribute : style attribute - owned by all elements

<p style="...."> hi</p>
